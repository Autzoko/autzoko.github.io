```python
import numpy as np

def fibonacci(n: int) -> list[int]:
    """Generate the first n Fibonacci numbers."""
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    return fib[:n]

print(fibonacci(10))
```

$$
\lim_{n \to \infty} \sum_{i=1}^{n} f(x_i) \Delta x = \int_a^b f(x) \, dx

## 繁體中文測試

設 $g(x)$ 為可微函數，根據泰勒展開：

$$
g(x) = \sum_{n=0}^{\infty} \frac{g^{(n)}(a)}{n!}(x-a)^n
