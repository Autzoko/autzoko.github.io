Vim Editor

Posted on 2024-06-11 In CS , tools

This notes is according to MIT’s course: missing-semester: Editors(Vim)

Vim is a popular command line editor and lots of visualized editors support vim emulation (like vs code). It’s designed for programmers by programmers.

Maybe not as fast as your previous editor, but stick on it for about 20 hours, you can master the vim.

Modes

Vim is a modal editor: multiple operating modes. Like writing, reading, etc.

When start vim:

  • Normal mode: all different key combinations behave in one way. Navigating.
  • Insert mode: Press key I to switch to insert mode from normal mode. By press key Escape to swtich back to normal mode. Type text in insert mode.
  • Replace mode: Press r to switch to replace mode. Forward and overwrite text.
  • Selection mode:
    • Visual mode: Press v to switch to visual mode.
      • Visual line: Press shift-v to switch to visual line.
      • Viusual block: Press ctrl-v to switch to visual block.
  • Command line mode: Press : (shift-;) to switch to command line mode.

You can rebind keys for more convenient use, like rebind the esacpe key to caps lock key for back to normal mode.

Navigating

Normal mode is used for navigate. And be an interfance to switch between other modes.

  • To move cursor up and down, left and right, use key hjkl, where j moves down, k moves up, h moves left and l moves right.
  • To move cursor by words, use key w and key b, where w moves forward by word and b moves backward by word. Use key e to move to the end of the word.
  • To move cursor by line, press 0 to move to the front of the line, press $ to move to the end of the line, press ^ to move to the first non-empty character of the line.
  • To scroll up and down, use ctrl-u to scroll up and ctrl-d to scroll down.
  • To move by buffer, press shift-g to move to the front and double press g to move to the very end.
  • To find the first specific character in current line, use command f to find that character. For example, find the first ‘c’ in current line use fc and find first ‘[‘ in current line use f[. To find the character before the cursor, use capital F to find.
  • To move cursor by given number of lines, add a number before typing the hjkl command: 4j means move down 4 lines, 6k means move up 6 lines.

For edit commands:

  • Press o to start a new line below the cursor.
  • Press u to undo and ctrl-r to redo.
  • Use d and combine other command to delete:
    • dw to delete a word when the cursor is at the front of a word.
    • de to delete to the end of the word from cursor position.
    • Double click d (dd) to delete the whole line.
  • Use c and combine other command to change:
    • ce change the end of the word and automatically switch to insert mode for rewrite.
    • Double click the c (cc) to change the whole line.
  • Use x to delete the character at cursor’s position.
  • Use r[charactor] to replace the character at cursor by new character [character].

For copy and paste:

  • Use y to copy ans p to paste.
    • yy for copying the whole line.
    • yw for copying the word.
  • To copy a block of codes, first press v to get into the visual mode and use hjkl to move the cursor to select codes, and press y to copy them. After immediately copying the code block vim will automatically back to normal mode and then you can use p to paste the codes you’ve just copied.
    • By pressing shift-v entering the visual line mode, each movement will select whole line.
    • By pressing ctrl-v entering the visual block mode, each movement will select a rectangular block of codes.
  • In visual mode, use pattern [number][move command] can select codes by moving cursor. For example: 3e means select the following three words in the visual mode, 7dw means delete seven words.

For searching:

  • Use / to search (or match) specific tokens. And press n for searching next matching item.
