About metaprogramming

Metaprogramming is a philosophy that refers to the process of programming, the problem about how to make us programming more efficiently, more scientific, etc. And how the program should be arranged, like its dependencies.

The first step to understand and implement metaprogramming is to learn Makefile.

Actually lots of tools that designed for making a program has been widely used, like maven for java, rustup for rust, etc.

Makefile

Makefile is a widely used build tool which can be written in script. By writing a Makefile and this script, the whole project can be compiled, linked and built simultaneously. The makefile specify a set of rules for programmers to follow to simplify the building process, including exception handle functions.

Currently, most makefile serve for C/C++ programs
